# Astro.py
Asteroid Detection in .fits files using Tensorflow Keras.

## How to detect
1. Organize your dataset into three folders
  ```
 >data
    >detect
      >set_of_fits
    >training
      >Asteroids
      >NotAsteroids
    >validation
      >Asteroids
      >NotAsteroids
  ```
  
  2. Set parameters according to your dataset in ```astro.py```
  
  3. Run ```python astro.py```
